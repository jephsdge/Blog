# 个人项目
## 1、项目描述

个人博客项目是一个前后端分离项目

个人博客的服务对象是个人(admin)，给个人提供编辑、发布、管理博客的服务，访客可以随意访问博客页面。所以后台管理只有一个账户，即管理员账户，负责对整个站点的管理
